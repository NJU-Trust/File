# 新交互规约

## 借款

### 消费类小额短期借款

```json
url:"loan/borrow1"

method:GET

param{

	"username":"str",

	”targetType“:"str",

	"targetName":"str",

	"beginDate:"str",

	"endDate":"str",

	"usageCategory":"str",//资金用途分类

	"usageDiscription":"str",//资金用途详述

	"usageEvidence":"str",//资金用途证明，应该是图片，（这里是图片路径）

	“money”:"double",

	"returnDate":"double",

	"rate":"double",

	"returnWay":"str",

}



//返回值是根据上面填写的借款标的生成的还款方案

return[

	{

		"difficulty":"int"//标识还款方案的难度，1星~5星

		“principal”:"double"//贷款本金

		"interest":"double"//利息

		“sum”:"double"//本息和

		“num”:"int"//还款期数，一次性还本付息为1期

		[{

			"s1":"double"//当期偿还的本息和

			“p1”:"double"//当期偿还的本金数

			“i1”:"double"//当期偿还的利息数

			“else”:"double"//剩余未偿还的本金数

		},

		.......

	]//该数组表示每期应还的钱，数组长度等于还款期数

}	

]
```

### 标的排序与筛选

- 若传递参数为空，则默认没有排序及筛选
- classification 含义可根据8.21给出的需求对号入座，或查看后端项目 nju.trust.entit.SmallProjectClassification 及 nju.trust.entit.LargeProjectClassification 查看详细解释

method : GET

{
    "low" : "double", // 百分比
    "high" : "double"

    "sort" : "money|startTime|interestRate|repaymentDuration(,asc|desc)",
